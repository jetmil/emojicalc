НЕДОСТАЮЩИЕ КОМБИНАЦИИ ЭМОЦИЙ (40 штук):

1. Радость + Грусть = Ностальгия/Сладкая грусть (светлая печаль)
2. Радость + Отвращение = Злорадство (радость от чужих неудач)
3. Радость + Предвкушение = Оптимизм (уже есть)

4. Доверие + Удивление = Любопытство (открытость к новому)
5. Доверие + Отвращение = Цинизм (доверие фактам + отвращение к методам)
6. Доверие + Гнев = Праведность (справедливый гнев)
7. Доверие + Предвкушение = Надежда (вера в лучшее будущее)

8. Страх + Грусть = Отчаяние (безнадёжность)
9. Страх + Отвращение = Паранойя (всё кажется опасным и отвратительным)
10. Страх + Предвкушение = Тревога (уже есть)

11. Удивление + Отвращение = Шок/Оторопь (неожиданное неприятное)
12. Удивление + Предвкушение = Ожидание чуда (противоположности - спонтанность vs планирование)

13. Грусть + Предвкушение = Меланхолия (грустное ожидание)

14. Отвращение + Предвкушение = Брезгливое ожидание/Опасение (предчувствие неприятного)

УСИЛЕНИЯ (одинаковые эмоции - уже реализовано):
- Радость + Радость = Эйфория
- Доверие + Доверие = Преданность
- Страх + Страх = Ужас
- Удивление + Удивление = Шок
- Грусть + Грусть = Депрессия
- Отвращение + Отвращение = Омерзение
- Гнев + Гнев = Ярость
- Предвкушение + Предвкушение = Нетерпение
